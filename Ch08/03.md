# حذف شارة (Tag)

لحذف شارة ما، يتم تنفيذ الأمر التالي
مع تغيير 
`<Tag-Name>`
باسم الشارة

<div dir="ltr">

`$ git tag -d <Tag-Name>`

</div>

لمزيد من المعلومات حول هذا الأمر ، يمكن زيارة الدليل الرسمي لـgit حول أمر
[git tag](https://git-scm.com/docs/git-tag)
